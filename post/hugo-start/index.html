Hugo搭建个人博客

2019年1月27日
| 软件
| 阅读

记录下博客搭建过程，网上虽然已有很多类似的教程但是都十分零碎的，本文主要记录本站的搭建过程。

在使用hugo之前用过hexo但是感觉hexo用起来很不方便，特别是在多台电脑写博时不是很方便，搭建环境需要安装NodeJS环境，比较而言hugo就一个单文件，启动什么的就简单多了，这也侧面体现了go语言的魅力。

安装

Hugo官方主页：https://gohugo.io/

  • 下载 在Hugo Releases上下载最新版的Release程序。

  • 配置环境变量

设置Windows系统环境变量，我的电脑右键–>属性–>高级系统设置–>在”高级”标签页中–>环境变量…，选中PATH变量，并编辑…，在编辑值中新添加hugo程序的路径即可，如：D:\Develop\hugo

  • 创建站点
hugo new site blog      #创建本地站点
  • 安装maupassant主题
cd blog
git clone https://github.com/rujews/maupassant-hugo themes/maupassant

常用命令

hugo new site blog      #创建本地站点
hugo new about.md       #生成关于页面
hugo new post/test.md   #创建文章
hugo server             #http://localhost:1313访问站点。
hugo										#生成静态文件 public

categories设置分类，tags设置文章标签。

---
title: "Hugo搭建个人博客"
date: 2019-01-27T20:55:18+08:00
draft: false
categories: ["软件"]
tags: ["hugo","博客"]
---

注意：默认创建的是草稿类型，需要将draft值改为false才能看到页面。

添加评论

默认支持disqus社会化评论，由于众所周知的原因disqus在国内并不可用。maupassant主题还支持utteranc评论是一种基于GitHub issue 评论官网在这utteranc.es，需要使用GitHub账号登录才行对于普通用户来说不是很合适。

所以这里添加来必力评论。

  • 修改maupassant/layouts/partials/comments.html，参考utteranc部分添加如下内容：
<!-- livere -->
{{ if .Site.Params.livere.enable }}
    <div id="lv-container" data-id="city" data-uid={{ .Site.Params.livere.livereid }}>
    <script type="text/javascript">
        (function(d, s) {
            var j, e = d.getElementsByTagName(s)[0];
            if (typeof LivereTower === 'function') { return; }
            j = d.createElement(s);
            j.src = 'https://cdn-city.livere.com/js/embed.dist.js';
            j.async = true;
            e.parentNode.insertBefore(j, e);
        })(document, 'script');
    </script>
    <noscript> 为正常使用来必力评论功能请激活JavaScript</noscript>
</div>
{{ end }}
  • config.toml添加配置
[params.livere]
  enable = true
  livereid = "此处填写livereid"

部署

在hugo项目目录执行hugo即可生成静态站点，此时只需要将public里的内容部署到Github Pages即可。

但是为了更方便的在其他项目中写作，建议新建个私仓将整个项目提交，public目录通过git submodule子项目管理。

  • git submodule使用介绍

其他

  • 域名绑定 在public目录新建CNAME文件内容为要绑定的域名，如本站域名devhu.win。使用https://www.cloudflare.com/ CDN 自带https。
