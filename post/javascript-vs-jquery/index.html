原生JavaScript与jQuery差异对比

2019年5月16日
| 技术
| 阅读

原生JavaScript与jQuery操作差异对比，基于w3schools.com整理。

HTML元素查找

  • 通过ID查找 >查找元素ID为id01的元素
#JS
var myElement = document.getElementById("id01");

#jQuery
var myElement = $("#id01");
  • 通过Tag查找 >返回所有<p>元素
#JS
var myElements = document.getElementsByTagName("p");

#jQuery
var myElements = $("p");
  • 通过Class Name查找 >返回所有class="intro"的元素
#JS
var myElements = document.getElementsByClassName("intro");

#jQuery
var myElements = $(".intro");
  • 通过CSS选择器查找 >返回所有class="Introo" <p>元素的列表。
#JS
var myElements = document.querySelectorAll("p.intro");    #`querySelectorAll`不支持IE8

#jQuery
var myElements = $("p.intro");

HTML元素操作

  • 设置文本内容 >设置元素的文本：
#JS
myElement.textContent = "Hello Sweden!";

#jQuery
myElement.text("Hello Sweden!");
  • 获取文本内容 >获取元素的文本：
#JS
var myText = myElement.textContent || myElement.innerText;

#jQuery
var myText = myElement.text();
  • 设置HTML内容 >设置元素的html内容：
#JS
var myElement.innerHTML = "<p>Hello World</p>";

#jQuery
var myElement.html("<p>Hello World</p>");
  • 获取HTML内容 >获取元素的html内容：
#JS
var content = myElement.innerHTML;

#jQuery
var content = myElement.html();

CSS操作

  • 隐藏HTML元素
#JS
myElement.style.display = "none";

#jQuery
myElement.hide();
  • 显示HTML元素
#JS
myElement.style.display = "";

#jQuery
myElement.show();

DOM操作

  • 删除元素
#JS
element.parentNode.removeChild(element);

#jQuery
$("#id").remove();
  • 获取父元素
#JS
var myParent = myElement.parentNode;

#jQuery
var myParent = myElement.parent();
